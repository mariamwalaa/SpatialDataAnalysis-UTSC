This data set comes from the UCI Machine Learning Repository and can be found here. According to the data abstract, this data set consists of red vinho verde wine samples from the north of Portugal. The purpose of this data set is to model wine quality based on physicochemical tests. This data set contains one integer variable and the remaining variables are doubles. There are 1,599 red wine samples and 12 variables describing each sample. There are no missing values in this data set. The dependent variable is wine quality.
